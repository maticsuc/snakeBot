### Requirements
- Pillow
- keyboard

```
pip install Pillow keyboard
```

### Instructions
Run the ```bot.py``` file and then open the game on your main monitor. Press enter and the bot should start playing the game.

### Disclaimer
Works best on 1920x1080 resolution (for now).
